# X浏览记录器

一个Chrome插件，用于记录和分析你的X(Twitter)浏览历史，了解推荐算法如何影响你看到的内容。

## 功能特点

### 📊 浏览统计
- 总浏览推文数
- 不重复推文数量
- 总浏览时间
- 平均会话时长

### 💬 互动分析
- 点赞、转发、回复、收藏统计
- 最常浏览的作者Top 10
- 互动率计算

### 🧠 算法洞察
- 内容类型分布（文本、图片、视频）
- 推广内容占比
- 平均浏览时长
- 算法特征分析

### 📜 历史记录
- 最近浏览的推文
- 浏览时长追踪
- 推广内容标记

## 安装方法

### 方法1：开发者模式安装（推荐）

1. 打开Chrome浏览器，访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `x-history-tracker` 文件夹
5. 安装完成！

### 方法2：添加图标

为了获得最佳体验，请为插件添加图标：

1. 在 `x-history-tracker` 文件夹中创建以下图标文件：
   - `icon16.png` - 16x16 像素
   - `icon48.png` - 48x48 像素
   - `icon128.png` - 128x128 像素

2. 可以使用任何图片编辑工具创建，或者从[Flaticon](https://www.flaticon.com)下载Twitter/X相关图标

## 使用方法

1. 访问 [x.com](https://x.com) 或 [twitter.com](https://twitter.com)
2. 正常浏览推文
3. 点击浏览器工具栏中的X浏览记录器图标
4. 查看统计和分析数据

## 数据隐私

- 所有数据仅存储在本地浏览器中
- 不会上传到任何服务器
- 不会收集个人身份信息
- 可随时清除所有数据

## 算法分析说明

### X推荐算法核心要素

1. **相关性评分** (Relevance Score)
   - 基于用户过去的互动
   - 推文质量和参与度
   - 关注关系和兴趣匹配

2. **内容类型偏好**
   - 视频内容获得更多曝光
   - 图片推文比纯文本更易传播
   - 回复和线程内容个性化推荐

3. **时间衰减因子**
   - 新内容优先
   - 热门内容延长生命周期
   - 用户活跃时间影响展示

4. **推广内容插入**
   - 大约5-15%内容为推广
   - 基于用户画像精准投放
   - 标记为"推广"的内容

本插件帮助你看清这些模式，了解算法如何塑造你的信息茧房。

## 技术实现

- **Manifest V3**: 使用最新的Chrome扩展API
- **Intersection Observer**: 追踪推文可见性和浏览时长
- **Local Storage**: 本地存储所有数据
- **Content Script**: 注入X页面提取数据
- **Service Worker**: 后台处理和存储
